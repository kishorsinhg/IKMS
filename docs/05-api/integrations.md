# Integrations

## IMAP Email Intake

V1 supports one or more configured IMAP intake mailboxes.

Configuration fields:

- Host
- Port
- TLS/SSL
- Username
- Password or app password
- Folder
- Polling interval
- Processed folder behavior

## AI Provider

V1 supports configurable AI provider settings, with Mistral Cloud as the first tested provider.

Configuration fields:

- Provider name
- API key/secret reference
- OCR model
- Chat model
- Embedding model
- Timeout and retry settings
- Enable/disable flags
